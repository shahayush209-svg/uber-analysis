# Uber Ride Analytics Dashboard

## Project Objective

The main objective of this project is to transform raw ride data into meaningful business insights through interactive visual analytics.

This dashboard is designed from a recruiter and business stakeholder perspective to showcase:

* Analytical problem-solving
* Data storytelling
* Dashboard development capabilities
* Business intelligence skills
* 

## Overview

This project is an interactive **Power BI dashboard** built to analyze Uber ride data and generate business insights related to bookings, customer trends, ride performance, and operational efficiency.

The dashboard helps stakeholders understand:

* Ride booking trends
* Revenue and trip performance
* Customer behavior
* Peak demand periods
* Ride category distribution
* Operational KPIs

This project demonstrates practical **Business Analytics**, **Data Visualization**, and **Dashboard Design** skills using Power BI.

## Dashboard Pages

### 1. Home Page

The home page acts as the navigation and introduction screen for the dashboard.

Features:

* Interactive navigation buttons
* Branding and dashboard overview
* User-friendly dashboard experience

### 2. Overview Dashboard

The overview page provides key business insights and KPIs.

Possible analytics included:

* Total bookings
* Revenue analysis
* Ride status analysis
* Distance and trip trends
* Booking value insights
* Peak hour analysis
* Vehicle category performance
* Customer usage trends
