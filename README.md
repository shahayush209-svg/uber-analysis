# Uber Analytics Dashboard – Power BI Project

## Overview

This project is an interactive **Uber Analytics Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into ride performance, trip trends, customer activity, and operational metrics through visually interactive reports.

The goal of this project is to demonstrate practical **Business Analytics** and **Data Visualization** skills by transforming ride-related data into meaningful business insights.

--

## Dashboard Pages

### 1. Home Page

The home page acts as a navigation and introduction screen for the dashboard.

Features include:

* Dashboard branding and UI design
* Navigation buttons/icons
* User-friendly layout for report access

### 2. Overview Dashboard

The overview page contains the core analytical visuals and KPIs.

Key insights include:

* Total trips analysis
* Ride trends over time
* Revenue and performance metrics
* Location-based insights
* Ride category analysis
* Operational performance indicators

---

## Tools & Technologies Used

* **Microsoft Power BI**
* **Power Query** for data cleaning and transformation
* **DAX (Data Analysis Expressions)** for calculations and KPIs
* Data Visualization techniques
* Interactive dashboard design
